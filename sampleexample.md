This is a placeholder entry in the `_resources` collection.

Resources are separate from the blog feed and show up on the
[Resources](/resources/) page. Each one gets its own page (with a table of
contents, via `toc: true` set in `_config.yml`), so they're well suited for:

- Finished study guides or outlines
- Sermon series overviews that link out to individual messages
- Reference material you want people to be able to find directly, without
  digging through the blog feed

To add a new resource, create a new Markdown file in `_resources/` with its
own front matter `title` and `excerpt`.
